```bash
$ CLIENT_ID=<apistore's id> AUTH_KEY=<auth key> npm test
```
or

to test what will request the apistore's real server
```bash
$ NOCK_OFF=true CLIENT_ID=<apistore's id> AUTH_KEY=<auth key> npm test
```
